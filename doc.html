Логика работы аналайзера следующая

если урл не NONE тогда..
1) биз айди если нет то..2
2) город если нет то..3
3) категории если нет то..4
4) бренд в городе если нет то..5 # убрали это условие
5) категория loc8 и штат с полным названием
в локейте название штата искать в нижнем регистре пробелы заменяя на дэфисы
если регулярка по поиску урла может возвращать индексные страницы(GET "/")
для этого при получении ''(пустой строки) с регулярки, return-им None
